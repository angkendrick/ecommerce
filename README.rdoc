== README
Ruby: 2.2.1
Rails: 4.2.3

Buy & Sell application
Built on Ruby on Rails and Bootstrap

Create User
User can sign-in
User can create a listing
User can upload an image (gets stored in dropbox)
User can 'buy' a listing of another User
User can view purchase history
